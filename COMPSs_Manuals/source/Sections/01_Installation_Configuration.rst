***************************************
|:gear:| Installation and Configuration
***************************************

This section will show you how to install the COMPSs framework in detail
with various mechanisms (*using pip* and *building from sources*) and
how to configure COMPSs for different computing environments
(e.g. HPC, Clusters, Clouds, etc.).

Following this section, you will be able to quickly set up your environment
and configure COMPSs for the best performance.
