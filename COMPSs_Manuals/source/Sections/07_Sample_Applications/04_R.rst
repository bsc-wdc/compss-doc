R Sample applications
=====================

The example in this section is a simple applications developed
in RCOMPSs to easily illustrate how to code and run COMPSs
applications. This application is executed locally and show different
ways to take advantage of all the COMPSs features.

The rest of the examples are more elaborated.

The codes of all the applications can be found under the
``/home/compss/tutorial_apps/r/`` folder.
